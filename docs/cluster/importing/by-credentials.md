# Importing a Cluster Using Master Node Credentials

If you have direct access to the master node from your local machine, you can import your existing cluster using its credentials. Follow the steps below:

- Select **I have the master node credentials** from the left panel of the **Import Cluster** page.  
![](/docs/images/final/import-cred.png)
- Choose your preferred SSH authentication method.  
- Provide the required credentials by uploading the files or entering the details manually.
