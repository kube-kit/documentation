# Importing a Cluster Using Master Node Credentials

If you have direct SSH access from your local machine where KubeKit is running, Then Follow the steps below:

- Select **I have the master node credentials** from the left panel and provide the master node credentials on the right panel.
![Import Credentials](/docs/images/final/import-cred.png)
- Fill **Host IP** and **SSH Username** of master node.
- Select master node **SSH** authentication method. Based on your selection, enter the **SSH** password or **SSH key** file.
