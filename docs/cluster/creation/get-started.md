# Getting Started

After launching KubeKit application, Click on the **`Get Started`** button, this will bring the below page.

![Create Cluster](/docs/images/final/create-cluster.png)

If you have any existing cluster cluster, you can proceed with [importing](/docs/cluster/importing/importing-a-cluster.md).
Now click **`Create Cluster`** button to start cluster creation, then follow the [requirement](/docs/cluster/creation/prerequisites.md) guideline.
