# Quick Start Guide
> KubeKit supports a wide range of operating systems, including Linux, Windows, and macOS, enabling you to deploy and manage Kubernetes clusters across diverse platforms with ease.

## Installing KubeKit

Before you start the installation, make sure your PC meets the minimum [**system requirements.**](/docs/installing-kubekit/system-req.md)

Now you can start installation process following [this guide](/docs/installing-kubekit/installing-kubekit.md) easily.

## Create a cluster

Creating a Kubernetes cluster with KubeKit is designed to be fast, simple and reliable. Anyone can create cluster with an intuitive GUI, no domain knowledge required.

You can start the cluster creation process by following [this guide](/docs/cluster/creation/creating-a-cluster.md).

## Import a cluster

Importing an existing kubernetes cluster from anywhere to kubekit is very simple.

You can import any existng kubernetes cluster by following [this guide](/docs/cluster/importing/importing-a-cluster.md) simply.

## Managing a cluster

**KubeKit** provides an intuitive interface to manage every aspect of your cluster, including nodes, workloads, networking, configurations, and access control, all within a unified dashboard.
You can manage your cluster by following [this manage guide](/docs/cluster/managing-cluster/managing-a-cluster.md).
