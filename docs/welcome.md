## What is KubeKit?

KubeKit is an innovative DevOps automation tool designed to simplify on-premises Kubernetes cluster creation and management. Traditionally, building a Kubernetes cluster required deep technical expertise, manual configuration, and extensive scripting. KubeKit transforms this complex process into an intuitive, GUI-driven experience empowering developers and DevOps engineers to deploy and manage clusters with ease, speed, and confidence. If you are new to KubeKit, you may visit the [Quick Start](./getting-started/quick-start-guide.md) guide for a faster onboarding experience.

### The Spark of an Idea

KubeKit was born out of a shared desire among engineers to simplify Kubernetes management. Inspired by tools like Lens that made Kubernetes navigation easier, the team aimed to go further  making on-prem cluster creation equally accessible. Building a Kubernetes cluster had always been a labor-intensive effort, demanding both time and a highly skilled team. The KubeKit team envisioned a tool that would automate this process entirely within a clean and user-friendly GUI.

### Overcoming Challenges

Creating a GUI for on-prem Kubernetes cluster building posed several challenges. The traditional process is manual, involving numerous scripts, variables, and dependencies that change frequently across OS and Kubernetes versions. KubeKit’s development team overcame these obstacles by consolidating all complexities into an automated, streamlined workflow. After multiple iterations and refinements, KubeKit emerged as a solution that turns an intricate process into a smooth and efficient experience.

### Vision

While tools like Lens laid the foundation, KubeKit’s vision extends further. It aims to be a comprehensive solution that not only builds clusters but also enables application deployment, monitoring, and management  all from one interface. The goal is to make Kubernetes accessible to everyone, removing the steep learning curve and the need for large operational teams. KubeKit is built with a mission to make Kubernetes management as intuitive as possible, for both beginners and seasoned professionals.

## Key Highlights

* **Zero code, all GUI:** Create on-prem clusters in minutes through guided workflows  no coding required.
* **Lesser steps, faster results:** Drastically reduce the manual steps typically involved in cluster setup.
