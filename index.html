Introduction

Build and manage on-prem Kubernetes clusters effortlessly with GUI

Introduction

Welcome to the official KubeKit documentation. This guide provides comprehensive resources to help you understand, deploy, and manage Kubernetes clusters using KubeKit.

KubeKit is an innovative, GUI-driven automation tool that streamlines on-premises Kubernetes cluster creation and management. It removes the need for deep technical expertise, manual setup, and complex scripting by providing a fast, intuitive, no-code experience for Developers, SysAdmins, and DevOps engineers. With KubeKit, you can easily create, configure, monitor and manage clusters while maintaining full control over their infrastructure. For a quick onboarding experience, visit the Quick Start guide.

For more information about KubeKit and to access the official website, visit kubekit.io.

Key Features

No-Code Cluster Creation Multi-Cluster Management
Fully GUI-based cluster creation and management. On-premises, bare-metal cluster support. Integrated SSL security. Multi-OS worker node support e.g., Ubuntu,Debian, CentOS, Redhat, Fedora and Amazon Linux nodes can run simultaneously. Learn more about Supported OS here. Import existing clusters using kubeconfig files or master node credentials. Unified management of multiple clusters from one dashboard.
GUI-Based Management Deployment Management
Manage and scale cluster resources without using command-line tools. Add or remove nodes effortlessly. Intuitive controls for complex Kubernetes operations. Simplified application deployment workflows. Built-in support for Helm package management.
Monitoring & Observability Troubleshooting Tools
Real-time monitoring of logs, metrics, and performance. CPU and memory usage tracking. Network traffic insights and fault detection. Visualization of storage usage and resource status. View resource descriptions and events. Access pod and deployment logs. Open terminals directly into nodes or pods.

Key Highlights

  • Zero code, Fully GUI based: Create on-prem clusters in minutes through guided workflows no coding required.
  • Lesser steps, faster results: Drastically reduce the manual steps typically involved in cluster setup.
  • Accessible for all: Designed for both new users and experienced Kubernetes administrators.

Benefits

KubeKit envisions transforming the Kubernetes ecosystem by creating a tool that:

  • Empowers all engineers from beginners to experts to build and manage clusters with minimal technical friction.
  • Combines diverse DevOps functions into one unified platform.
  • Reduces time, complexity, and operational costs.
  • Builds a strong, collaborative community around accessible Kubernetes innovation.

With KubeKit, on-premises Kubernetes management becomes faster, simpler, and more intuitive enabling teams to focus on what truly matters: building and scaling their applications.
