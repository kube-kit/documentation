# Introduction

>Welcome to the official KubeKit documentation. This guide provides comprehensive resources to help you understand, deploy, and manage Kubernetes clusters using <a href="https://kubekit.io">**KubeKit**</a>.


[KubeKit](https://kubekit.io) is an innovative, GUI-driven automation tool that streamlines on-premises Kubernetes cluster creation and management. It removes the need for deep technical expertise, manual setup, and complex scripting by providing a fast, intuitive, no-code experience for Developers, SysAdmins, and DevOps engineers. With KubeKit, you can easily create, configure, monitor and manage clusters while maintaining full control over their infrastructure. For a quick onboarding experience, visit the [Quick Start](/docs/getting-started/quick-start-guide.md) guide.

| **No-Code Cluster Creation**     | **Multi-Cluster Management** |
|----------------------------------|------------------------------|
| Fully GUI-based cluster creation and management. On-premises, bare-metal cluster support. Integrated SSL security. Learn more about Supported OS [here](/docs/cluster/creation/prerequisites.md). | Import existing clusters using kubeconfig files or master node credentials. Unified management of multiple clusters from one dashboard. |

| **GUI-Based Management**       | **Deployment Management**   |
|--------------------------------|-----------------------------|
| Manage and scale cluster resources without using command-line tools. Add or remove nodes effortlessly. Intuitive controls for complex Kubernetes operations.| Simplified application deployment workflows. Built-in support for [Helm](/docs/cluster/managing-cluster/helm.md) package management.|


| **Monitoring & Observability**   | **Troubleshooting Tools**    |
|----------------------------------|------------------------------|

Open [terminals](/docs/cluster/managing-cluster/terminal-access.md) directly into nodes or pods.|



## Key Highlights

* **Zero code, Fully GUI based:** Create on-prem clusters in minutes through guided workflows  no coding required.
* **Lesser steps, faster results:** Drastically reduce the manual steps typically involved in cluster setup.
* **Accessible for all:** Designed for both new users and experienced Kubernetes administrators.
